## Dokumentációs szabály

A doksik legyenek rövidek, tulajdonolhatók és futtathatók. Egy erős belépési
pont jobb, mint minden mappában ugyanazt összefoglaló mirror.

Magyar társ-dokumentum csak a legfontosabb résztvevői doksikhoz kell:
orientáció, onboarding, workshop feladat és fontos solution belépési pontok.
